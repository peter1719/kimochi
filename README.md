# kimochi


接線配置
----------------------------------------------------

- 藍芽模組 HC-05

    TX   -->  D8   
    RX   -->  D9  
    VDD  -->  5V  
    GND  -->  GND  


- 旋鈕模組

    CLK  -->  D3  
    DT   -->  D4  
    SW   -->  D5  
    VDD  -->  5V  
    GND  -->  GND    



- PPG心律模組 XD-58C

    SW   -->  A0  
    VDD  -->  D10  
    GND  -->  GND   



- 陀螺儀 MPU9250

    SDA  -->  A4  
    SCL  -->  A5  
    VDD  -->  5V  
    GND  -->  GND  



- WS2812  the number of led can 
  be set at Adafruit_NeoPixel(number,...) (line 40)

    DIN  -->  D6  
    VDD  -->  5V  
    GND  -->  GND   


- 按鈕

    \+     -->  D5  
    \-     -->  GND  

- 震動馬達

    \+     -->  D11  
    \-     -->  GND
